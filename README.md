## 关于本项目 

使用Laravel框架及生态开发出来的Tunnel Broker（隧道中间人）面板，使您能够通过只有IPv4连接并支持IPv6的主机或路由器建立隧道，实现访问IPv6 Internet。



本项目部分代码来自MercyCloud Automation自动化服务及Core核心服务

## 支持隧道

IPV6 隧道

IPV6 BGP 隧道



## 特点

* 自动更新AS-SET（使用RIPE NOC API）
* 自动创建RDNS记录
* 自动配置FRR BGP
